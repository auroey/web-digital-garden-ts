title: Type System
date: 2025-08-13
---

**目的**：

- 把渲染器核心组件（BSDF、光源、体积、积分器等）写成**与具体数值类型无关**的“泛型版本”。
    
- 通过**类型别名 + 模板参数**决定实际使用哪种类型，从而快速切换不同渲染模式。
    

**好处**：

- **可组合**：比如你可以同时打开“偏振光 + GPU JIT + 可微分”功能，类型系统会在编译时组合生成一个新的渲染器版本。
    
- **零运行时开销**：所有决策在编译期完成，运行时不需要分支判断。
    
- **减少代码重复**：不需要维护多个几乎相同的渲染器版本。
