# JavaScript基础

## 概述

JavaScript 是一种动态编程语言，主要用于网页开发。

## 基本概念

### 变量
```javascript
let name = "Quartz";
const version = "4.5.1";
```

### 函数
```javascript
function greet(name) {
    return `Hello, ${name}!`;
}
```
